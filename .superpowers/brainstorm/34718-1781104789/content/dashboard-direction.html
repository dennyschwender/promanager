What should the Dashboard become?

The current dashboard shows the same "next 5 events" table as All Events, plus quick action links. Here are 3 alternative directions — each focuses on data that exists but has no other page in the app.

Option A: Team Pulse (Recommended)

A single-pane overview for coaches and players showing the team's health:

Widgets

  • Attendance rate this season — team-wide % with trend arrow
  • Upcoming events — compact list (no table), just countdown + title
  • Pending responses — who hasn't replied yet, grouped by event
  • Injured / absent — all players currently marked absent or injured, in one place
  • Recent chat activity — last 5 messages across all events
  • Consecutive absences alert — players who missed 3+ events in a row

Trade-offs

  • Requires writing new aggregation queries
  • Most useful info for coaches; players may care less about team-wide stats

Option B: My Season

A personal dashboard for the logged-in player (or coach viewing themselves):

Widgets

  • My attendance rate — personal % with breakdown by event type (trainings vs matches)
  • Unread notifications — latest 3 with previews
  • Next event — single card with my current status + quick-update button
  • My absences — upcoming period absences or recurring rules active now
  • My notes — recent attendance notes I've added

Trade-offs

  • Overlaps with My Report page (which already shows personal stats)
  • Coaches care more about the team than themselves

Option C: Season Command Center

A rich admin/coach view with actionable widgets:

Widgets

  • Season progress bar — "Day 47 of 90" with event count
  • Attendance leaderboard — top 5 / bottom 5 players by rate
  • Upcoming events — as compact cards with attendance summary badges
  • Player availability grid — a small matrix of next 5 events x 10 players, color-coded
  • Lowest-attendance events — events with the most unknowns
  • Watch list — players on injury/absence streak

Trade-offs

  • Most complex to build
  • Grid might overwhelm on mobile
  • Best for coaches; less useful for players
